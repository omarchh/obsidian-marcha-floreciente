Creadas por [[Mushbloom Enterprises]], estas capsulas de vacio se utilizan para poder guardar hierbas de todo tipo y conservarlas como si se encontraran plantadas en un suelo optimo para ellas. Hay de diversos tamaños, desde las mas pequeñas de bolsillo para pequeñas plantas como treboles, hasta capsulas de tamaño de un cofre, pudiendo preservar arbustos pequeños como rosales y arbustos de fruta.
## Descripcion Fisica
La mayoria de las capsulas estan conformadas por una cupula de vidrio y una base magica creada a partir de sabia de arbol, mana y algunos otros aditamentos para magicamente proveer nutrientes a las plantas. Tienen un compartimento especial que permite agregar diferentes nutrientes para experimentar con plantas.
## Funcion especifica
Fueron principalmente diseñadas para el uso cotidiano en tiendas y restaurantes donde se mantenian siembras pequeñas para poder tener alimento fresco. Cuando la marcha floreciente se desato, Haven pidio a Mushbloom Enterprises crear capsulas mas pequeñas para poder guardar "Flores" como regalos para la nobleza. Estas capsulas permiten que las plantas vivan durante un largo tiempo siempre y cuando se les agregue nutrientes de vez en cuando.
